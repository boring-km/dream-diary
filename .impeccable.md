# 그림드림 — 디자인 컨텍스트

> 꿈 기록 아카이브. 손그림 종이 카드를 서랍에 모으고, 흩어진 더미에서 잊은 꿈을 "발굴"하는 담백한 서비스.
> 이 문서는 impeccable 및 모든 디자인 작업의 기준점이다.

## Design Context

### Users
- **누구**: 개인. 자기 꿈을 기록·소장하려는 한국어 사용자.
- **맥락**: 새벽/잠결에 떠오른 꿈을 빠르게 붙잡고(골든타임), 낮·나중에 다시 뒤적이며 잊은 꿈을 우연히 만난다.
- **할 일(JTBD)**: ① 마찰 없이 꿈을 붙잡아 보관 ② 쌓인 꿈을 발굴하며 다시 음미 ③ "내 37번 꿈" 처럼 소장·호명.
- **태도**: 여유·사적·"느좋". 생산성 도구가 아니라 **추억 상자**.

### Brand Personality
- **3단어**: 담백한 · 은밀한 · 손그림 같은(handmade).
- **정서 목표**: 차분함, 발굴의 잔잔한 설렘, 소장의 뿌듯함. 압박·자극·경쟁심 0.
- **톤**: 조용하고 정돈됨. 말 적게, 여백 많이. 좋은 말 위주(평가문구), 담백한 한 문장 피드백.

### Aesthetic Direction
- **방향**: 미니멀 플랫 (refined minimal). 정돈된 인쇄체 + 손그림 일러스트의 대비.
- **테마**: **페이퍼 라이트 통일.** 따뜻한 오프화이트 종이 바탕. (다크 없음)
- **색**: **파란 잉크 단색 절제.** 거의 모노 — 파랑(잉크) + 종이. 60-30-10, 파랑은 희소하게. 감정 표현도 색 남발 대신 형태/아이콘 우선.
- **글자**: 깔끔한 인쇄체만 (손글씨체 금지). 개성은 글자가 아니라 손그림에서.
- **질감**: 미니멀 플랫 — 종이 텍스처/잉크 번짐 최소. 물성은 *질감*이 아니라 *인터랙션*(서랍 열림, 카드 뒤적임)으로 표현.
- **시그니처**: 파란 손그림 종이 카드 + 카드마다 고유번호 + 그날의 손그림 도장 아이콘. (출처 이미지: 파란 라인 드로잉 / 납작한 파란 우표 일러스트)

### Anti-references (이렇게 보이면 실패)
- 식물 성장 웰니스 앱 클리셰 (Forest / Finch / Habitica 류) — 흔함.
- AI 슬롭: 보라→파랑 그라데이션, 네온 강조, 글래스모피즘, 그라데이션 텍스트, 카드 좌측 컬러 스트라이프.
- 생산성 대시보드 / 정렬된 DB 느낌의 깔끔한 리스트·그리드.
- 화려·과자극·시끄러움. 눈 아픈 대비.

### Design Principles
1. **발굴 > 검색.** 의도된 비효율이 정체성. 꿈은 어렴풋·흩어짐 — 정렬된 그리드 금지. 검색은 숨은 비상구(작은 돋보기)일 뿐.
2. **손그림이 유일한 장식.** 글자는 정돈된 인쇄체, 개성·온기는 손그림 일러스트/아이콘으로만.
3. **파랑 1색 + 종이.** 색은 희소할 때 힘이 난다. 감정도 색 남발 X — 형태·아이콘·테두리 우선.
4. **담백·여백.** 압박 0, 한눈에, 눈 안 아프게. 미니멀 플랫. 말은 적게.
5. **물성은 동작으로.** 서랍 열림·카드 뒤적임·"오늘의 발굴" 같은 인터랙션으로 아날로그 손맛. 텍스처 떡칠 아님.

### 제품 구조 메모 (디자인이 따라야 할 골격)
- **메타포**: 서랍 단독. 3~4개 고정, 위치=깊이=비밀도(위=자주 꺼냄 / 아래=묻어둠).
- **내용물**: 서랍 안에 흩어진 손그림 종이 카드 더미(마구잡이, id 기반 고정 배치). 일부만 보임 → 뒤적여야 봄.
- **카드**: 손그림 + 고유번호(유저별 1부터, 영구·불변·결번 유지) + 그날 손그림 아이콘.
- **자기보상**: "오늘의 발굴"(랜덤 1장), 수집 카운터/뱃지.
- **검색**: 구석 작은 손그림 돋보기, 본문 매칭, 평소 안 드러남, 닫으면 흔적 0.
